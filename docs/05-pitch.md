# Pitch: MILA 💰 - O Assistente Para Suas Microdespesas

> **Duração:** 3 minutos | **Público:** Investidores, usuários finais, stakeholders

---

## Roteiro Sugerido

### 1. O Problema (30 seg) 🎯

**"Você sabe para onde vai seu dinheiro?"**

Pequenas despesas passam despercebidas: um café de R$5, um lanche de R$8, um aplicativo de R$9. Isoladas, parecem insignificantes. Mas aquele cafezinho diário? **R$150/mês**. O lanche da semana? **R$350/mês**. Aquele app esquecido? **R$120/mês**.

**No final do mês, você perdeu R$600+ sem nem perceber.**

Pesquisa mostra que **67% das pessoas não sabem para onde vai seu dinheiro** com esses pequenos gastos. Elas querem controle, mas não têm uma ferramenta simples e amigável para rastrear.

---

### 2. A Solução (1 min) 💡

**Apresente: MILA - Miniatura de Gastos Inteligente**

MILA é um assistente conversacional que:

1. **Captura gastos em linguagem natural**: Você diz "Gastei R$5 com café" e MILA registra automaticamente
2. **Aprende seus padrões**: Identifica quanto você gasta com café, lanches, transporte, etc.
3. **Oferece insights acionáveis**: "Você gasta R$120/mês com café. Se reduzisse para 3x por semana, economizaria R$180!"
4. **Mantém você no controle**: Sempre baseado em dados reais, nunca inventa informações

**O diferencial?** MILA é **simples, amigável, e 100% focada em microdespesas**. Não é um app pesado de banco. É um amigo que te ajuda a enxergar onde seu dinheiro vai.

---

### 4. Diferencial e Impacto (30 seg) 🚀

**Por que MILA é diferente:**

1. **Foco Ultra-Específico**: Não trata investimentos, financiamento, seguros. Apenas microdespesas.
2. **100% Anti-Alucinação**: Nunca inventa números. Tudo baseado em dados confirmados.
3. **Conversa Natural**: Você não aprende um sistema novo. Conversa como com um amigo.
4. **Educativo, não crítico**: Nunca julga seus gastos, apenas oferece dados para decisão.

**Impacto potencial:**
- **Para o usuário**: Economizar R$100-300/mês (apenas reduzindo microdespesas)
- **Para a sociedade**: Criar consumidores **mais conscientes e empoderados**
- **Para fintechs**: Integrar MILA para **aumentar engajamento** e retenção de usuários
